# React Sample App

* React, React Router
* Webpack, Babel, ESLint(Airbnb)
* Jest, enzyme
* Sass, css modules, Bootstrap

# Start dev
* ```npm install```
* ```npm start```
* ```npm run test```
* ```npm run converage```
* ```npm run lint```

# Deployment
* ```npm run build```

Builds the app for production to the build folder.
