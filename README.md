## How to run the app

1.  Install latest Node
2.  Checkout this repository
3.  `npm install`
4.  `npm run build`
5.  `npm start`

The app is now running on default HTTP port on all the IPs.

If the last step fails, change the port in [`server.cfg.json`](/server.cfg.json) file.
